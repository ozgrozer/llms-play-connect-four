# LLMs Play Connect Four

This is a **Next.js**, **Tailwind CSS**, and **Shadcn** project.

Copy the `.env.example` file to `.env` and add your API credentials.

## Development

Install dependencies

```
pnpm install
```

Start development

```
pnpm dev
```

Build the app

```
pnpm build
```

Start the app

```
pnpm start
```
